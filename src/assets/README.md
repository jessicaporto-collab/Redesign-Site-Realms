# Assets

Place your media files here:

- `images/` — logos, hero images, product screenshots
- `icons/` — SVG icon files
- `video/` — hero background videos (e.g. hero.mp4)

In production, replace the sample video URL in `src/pages/Home.jsx` with:
```jsx
<source src="/src/assets/video/hero.mp4" type="video/mp4" />
```
